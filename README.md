# developershub-internship
Data Science Internship Tasks - DevelopersHub

## 📌 Overview

This repository contains 5 machine learning and data analysis tasks completed during my internship at Developshub Corporation.  
The projects cover data exploration, classification, regression, and business insight extraction using real-world datasets.

---

## 📂 Projects Included

### 📊 Task 1: Iris Dataset Analysis
- Explored dataset structure using Pandas
- Visualized relationships using scatter plots, histograms, and box plots
- Identified patterns between flower species features

**Skills:** Data visualization, EDA, data summarization

---

### 💳 Task 2: Credit Risk Prediction
- Handled missing values in loan dataset
- Performed exploratory data analysis (income, loan amount, education)
- Trained Logistic Regression / Decision Tree model
- Evaluated using accuracy and confusion matrix

**Skills:** Classification, data cleaning, model evaluation

---

### 🏦 Task 3: Customer Churn Prediction
- Encoded categorical features (Gender, Geography)
- Built classification model to predict customer churn
- Analyzed feature importance (Age, Balance, Activity status)

**Skills:** Encoding, supervised learning, feature importance

---

### 🏥 Task 4: Insurance Claim Prediction
- Built Linear Regression model to predict medical charges
- Analyzed impact of BMI, Age, and Smoking on insurance cost
- Evaluated using MAE and RMSE

**Skills:** Regression modeling, error analysis, feature correlation

---

### 📞 Task 5: Personal Loan Acceptance Prediction
- Explored customer demographics (age, job, marital status)
- Built classification model (Logistic Regression / Decision Tree)
- Identified customer groups more likely to accept loans

**Skills:** Business analytics, classification, insight extraction

---

- Handling categorical and missing data effectively
