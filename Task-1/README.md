# 🌸 Iris Dataset - Exploratory Data Analysis

## 📌 Project Overview

This project performs Exploratory Data Analysis (EDA) on the Iris dataset to understand patterns, relationships, and feature importance across different flower species.

## 📊 Dataset

The dataset contains measurements of iris flowers:

* Sepal Length
* Sepal Width
* Petal Length
* Petal Width
* Species (Setosa, Versicolor, Virginica)

## 🛠️ Tools & Libraries

* Python
* Pandas
* Matplotlib
* Seaborn

## 📈 Analysis Performed

* Data inspection and summary statistics
* Scatter plot to analyze feature relationships
* Histograms to understand distributions
* Box plots to detect spread and outliers
* Pairplot for multi-variable analysis

## 🔍 Key Insights

* The dataset is balanced with no missing values
* Petal length and width are the most important features for distinguishing species
* Setosa species is clearly separable from others
* Versicolor and Virginica show some overlap
* Sepal features are less effective for classification
